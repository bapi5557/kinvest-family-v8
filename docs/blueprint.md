# **App Name**: Kincash

## Core Features:

- Family Authentication Gateway: Secure single-login system for all family members utilizing a shared credential via Firebase Auth.
- Household Registry: Manage the list of participating family members with full add, edit, delete, and search capabilities.
- Multi-Category Ledger: Input and track expenses across essential family categories like groceries, rent, LPG, and medical needs.
- Smart Expense Tool: Generative AI analysis tool to provide insights on monthly spending patterns and suggest potential savings based on historical data.
- Live Expense Dashboard: Real-time summary of family financial health with data synchronization via Firestore across all devices.
- Financial Analytics Suite: Visual reporting engine that aggregates spending into monthly and yearly breakdowns.
- Document Export Studio: Generate professional PDF reports of family spending to share via local communication channels.

## Style Guidelines:

- Primary color: Deep Slate Blue (#395B96) to evoke stability and trust.
- Background color: Dark Onyx (#1E232E) for a premium, agency-style high-contrast interface.
- Accent color: Celestial Cyan (#85DAE8) to provide vibrant, high-contrast highlights for call-to-actions.
- Headline font: 'Space Grotesk' (sans-serif) for its technical and modern personality. Body font: 'Inter' (sans-serif) for optimal legibility of complex data tables.
- Sharp, 2px stroke-weight geometric icons used consistently across expense categories and navigation.
- A compact, card-based mobile-first design centered on ease of use for quick data entry on-the-go.
- Gentle fade-in transitions for dashboard cards and horizontal slide-in effects for member profile details.
